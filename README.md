# 🚖 Real-Time Ride Analytics Pipeline

An end-to-end Data Engineering project that demonstrates how to build a real-time analytics pipeline using Azure Event Hub, Apache Spark Structured Streaming, and a Star Schema data warehouse.

The project simulates ride events generated by a web application, streams them through Azure Event Hub, processes the data in real time using Apache Spark, applies data validation and transformations, and loads the processed data into a Star Schema for analytical reporting.

---

## 📖 Project Overview

Modern ride-sharing platforms generate millions of events every day. Traditional batch processing introduces delays that make real-time monitoring and analytics difficult.

This project demonstrates how streaming technologies can be combined to build a scalable and reliable real-time data pipeline capable of processing ride events continuously.

---

## 🎯 Objectives

- Build an end-to-end real-time data pipeline.
- Ingest streaming events using Azure Event Hub.
- Process streaming data with Apache Spark Structured Streaming.
- Validate and clean incoming data.
- Apply business transformations.
- Load processed data into a Star Schema.
- Enable fast analytical SQL queries.
